================================================================================
MATHEMATICAL FORMULAS DOCUMENTED
================================================================================

1. Equal-area Projection (Schmidt):
   r = √2 × sin(θ/2)
   X = r × cos(φ), Y = r × sin(φ)

2. Equal-angle Projection (Wulff):
   r = tan(θ/2)
   X = r × cos(φ), Y = r × sin(φ)

3. Color Normalization:
   v_norm = (v - v_min)/(v_max - v_min) × (c_max - c_min) + c_min

4. Colormap Midpoint (Asymmetric):
   midpoint = 1 - v_max/(v_max + |v_min|)

5. Color Interpolation:
   C(t) = C1 × (1-t) + C2 × t

================================================================================
TROUBLESHOOTING COVERED
================================================================================

Common Issues:
  ✓ Empty plot → Call plotProj() first
  ✓ Low resolution → Increase DPI to 600
  ✓ Banded colormap → Increase cmapbins to 256+
  ✓ Cropping fails → Install wand library
  ✓ Wrong projection → Check ProjType spelling
  ✓ Directions not showing → Use [[u,v,w]] format
  ✓ Colorbar missing → Call plotColorbar()
  ✓ Memory error → Reduce nump value
  ✓ Slow plotting → Lower nump for development

Performance Tips:
  ✓ Preview: nump=101 (fast)
  ✓ Normal: nump=201-301 (balanced)
  ✓ Publication: nump=501+ (high quality)
  ✓ Cache colormaps for reuse
  ✓ Batch export multiple formats
